  == Development Environment ==
  * KEILMDK Version 5.12 or above.
  * Programmer/Debugger: ST-LINK V2
  * Programming/Debugginginterface: JTAG/SWD
  
  == How to use ==
  1. Put the jumpers on LED JMP.
  2. Press and hold the buttons:
    * Joystick CENTER --> All LEDs on.
    * Joystick A --> LED1 on.
    * Joystick B --> LED2 on.
    * Joystick C --> LED3 on.
    * Joystick D --> LED4 on.
